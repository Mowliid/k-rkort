QuranWebApp v10 - bättre svensk röst

Nytt:
- Väljer automatiskt bästa svenska rösten i webbläsaren.
- Prioriterar Natural/Online/Neural/Microsoft-röster om de finns.
- Läser svenska långsammare: 0.80x som standard.
- Du kan själv välja svensk röst och hastighet i rutan högst upp.
- Parentes-kommentarer tas bort vid uppläsning så rösten låter mer naturlig.

Tips:
Använd Microsoft Edge på Windows. Den brukar ha bättre svenska röster än Chrome.
För riktigt mänsklig AI-röst krävs en extern AI voice API, men då behövs API-nyckel och kostnad.
