STARTA APPEN

På GitHub Pages:
1. Ladda upp alla filer och mappar precis som de ligger i zip-filen.
2. Öppna din GitHub Pages-adress.
3. Välj en sura och tryck på den stora play-knappen.

På datorn:
1. Öppna mappen i Visual Studio Code.
2. Öppna Terminal och skriv: node server.js
3. Öppna http://localhost:3000

Alla 114 svenska MP3-filer finns redan i mappen surah-audio.
