ENKEL FÖRKLARING AV VARJE VERS

Den här versionen har en knapp under varje vers:
"Enkel förklaring"

När du klickar får du versen omskriven på enklare svenska.

Start:
1. Öppna QuranWebApp_Matrood i VS Code.
2. Öppna terminal.
3. Skriv: node server.js
4. Öppna: http://localhost:3000

Ingen API-nyckel behövs.
Inget npm install behövs.
